The Skeleton-skeleton Kitchen Sink

The Skeleton-skeleton is a Gulp/LESS toolchain for Skeleton: A Dead Simple, Responsive Boilerplate for Mobile-Friendly Development, based on Skeleton-less.

Use the Skeleton-skeleton toolchain together with this Kitchen Sink page to easily customize Skeleton. The Skeleton-skeleton uses BrowserSync allowing you to develop your responsive layouts on multiple devices at once.

If you need some help hit up the Skeleton documentation. You can find the Skeleton-skeleton on Github.
